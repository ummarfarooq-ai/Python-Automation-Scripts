# Python Automation Scripts

This repository contains simple Python scripts designed to automate everyday tasks.

The scripts demonstrate how Python can be used to save time by automating repetitive operations.

## Features

- File organization automation
- Secure password generation
- Automated email sending

## Project Files

file_organizer.py – Automatically organizes files into folders  
password_generator.py – Generates strong random passwords  
email_sender.py – Sends emails using SMTP

## Technologies Used

- Python
- Built-in Python libraries (os, smtplib, random, string)

## Installation

1. Clone the repository

git clone https://github.com/yourusername/Python-Automation-Scripts.git

2. Run scripts

python password_generator.py

## Purpose

This project demonstrates practical Python automation techniques for beginners and developers.
